Sonos Play URI
==============

Install:

    $ virtualenv spu
    $ cd spu
    $ ./bin/source/activate 
    $ git clone https://github.com/Adniel/SonosPlayUri.git
    $ pip install flask soco
    $ cd SonosPlayUri
    $ python socoplayer.py


Surf to:

    http://localhost:5001/

Option:

    http://localhost:5001/?programid=2901

where ``progamid``is SR program id
